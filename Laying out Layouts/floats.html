FLOATS

Important Aspects

  • an element can float to the left or right side of its container (or not at all)
  • an element with position value "absolute" ignores the float property
  • elements (such as text/inline elements) next to a floating element flow or wrap around it

Uses

  • to wrap any inline elements around a defined HTML element such as: lists, paragraphs, divs, spans, tables, iframes, and blockquotes
  • to format and position content on a page

Examples of Code (CSS)

img {
    float: left;
}

img {
    float: right;
}

img {
    float: none;
}
