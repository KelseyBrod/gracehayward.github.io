FLEXBOX

Important Aspects

  • designed for one-deimensional layout
  • makes elements responsive - they "flex" to fit

Uses

  • to set space distribution between elements
  • to control dimensions and allignment of items

Why is it necessary?

So that code can be visible and accessable on varying display devices and screen sizes

This page employs the flexbox model - if you adjust the height and width of the screen, the columns and grids respond!!!
